# Chat Application (Spring Boot + JavaFX)

Това е разпределено чат приложение, реализирано с:
- **Spring Boot** – сървърна част (REST + WebSocket)
- **JavaFX** – клиентска част
- **H2 Database** – вградена база данни
- **Maven** – управление на зависимости и стартиране

Приложението поддържа **real-time комуникация** между множество клиенти чрез WebSocket.

---

## Изисквания

- Java JDK 23
- Maven
- Git (по желание)
- Интернет или локална мрежа (LAN) за комуникация между устройства

---

## Стартиране на приложението

Приложението се състои от **две части**:
1. Сървър (Spring Boot)
2. Клиент (JavaFX)

### 1️⃣ Стартиране на сървъра

Сървърът се стартира чрез Spring Boot main класа: com.example.chat.ChatApplication

#### Начин 1: От IDE (препоръчително)
- Отворете проекта в IntelliJ IDEA
- Намерете класа `ChatApplication`
- Стартирайте го с **Run**

След успешно стартиране сървърът ще бъде достъпен на: http://localhost:8080

H2 конзолата е достъпна на: http://localhost:8080/h2-console

---

### 2️⃣ Стартиране на клиент (JavaFX)

Клиентът се стартира **отделно** от сървъра.

Отворете терминал в основната директория на проекта (където се намира `pom.xml`) и изпълнете:

```bash
mvn clean javafx:run
```
Всеки стартиран клиент се свързва към сървъра и може да комуникира в реално време.

--- 

## Архитектура

- Spring Boot сървър
- WebSocket endpoint за реално време
- REST API за управление на потребители и разговори
- H2 база данни за съхранение на данни
- JavaFX клиент
- Графичен интерфейс
- WebSocket клиент за изпращане и получаване на съобщения

---

## Забележки

1. Сървърът трябва да бъде стартиран преди клиентите

2. За свързване от други устройства в мрежата, клиентът трябва да използва IP адреса на сървъра вместо localhost
Качих протипна версия с регистрация/логин/чат/ потребителски интерфейс. остава да направя лист с приятели/нотификиции/покани и групи.
